# KaloriKu - Smart Nutrition Tracking System

Selamat datang di *repository* organisasi **KaloriKu**! 🚀

Kami adalah tim yang berdedikasi untuk membangun aplikasi pelacakan kalori dan nutrisi cerdas. KaloriKu tidak hanya sekadar mencatat makanan, tetapi memanfaatkan kekuatan *Artificial Intelligence* (AI) dan *Machine Learning*—terutama *Natural Language Processing* (NLP) dan model regresi—untuk memberikan analisis nutrisi yang presisi dan personal kepada pengguna.

## 🛠️ Tech Stack & Architecture

Untuk memastikan performa, skalabilitas, dan akurasi analisis, kami menggunakan ekosistem teknologi berikut:

*   **Front-End:** Next.js 15, React, Tailwind CSS v4, TypeScript.
*   **Back-End:** (Sesuaikan dengan framework pilihan, e.g., Node.js / FastAPI / Go).
*   **Data Science & AI:** Python, NLP Libraries (HuggingFace/Spacy), Scikit-Learn/TensorFlow (untuk model regresi).
*   **Infrastruktur & Kolaborasi:** GitHub Actions (CI/CD), Model Context Protocol (MCP) untuk integrasi AI.

## 📝 Development Guidelines

Sebagai panduan kolaborasi tim, kita menerapkan standar berikut dalam setiap siklus *sprint*:

1.  **Issue Tracking:** Semua tugas (Fitur, Bug, atau *Chore*) **wajib** dibuatkan *Issue* terlebih dahulu sebelum mulai *coding*.
2.  **Branching Naming Convention:**
    *   `feat/nama-fitur` (Untuk fitur baru)
    *   `fix/nama-bug` (Untuk perbaikan bug)
    *   `chore/nama-tugas` (Untuk *update dependencies*, *refactoring*, dll)
3.  **Pull Requests (PR):**
    *   Sertakan referensi nomor *Issue* di deskripsi PR (contoh: `Closes #12`).
    *   Minimal butuh **1 *Approve*** dari *engineer* lain sebelum bisa di-*merge* ke *main branch*.
4.  **Definition of Done (DoD):** Kode bisa dianggap selesai jika sudah memenuhi kriteria fungsional, tidak ada konflik saat integrasi, dan lulus uji coba pada *environment staging*.

Mari kita bangun sistem ini dengan rapi dan terukur! 💻✨
